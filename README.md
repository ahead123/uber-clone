## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Create a .env file

   ```bash
    EXPO_PUBLIC_CLERK_PUBLISHABLE_KEY=
    DATABASE_URL=
    EXPO_PUBLIC_SERVER_URL=
    XPO_PUBLIC_GEOAPIFY_KEY=
    EXPO_PUBLIC_GOOGLE_API_KEY=
   ```

3. Start the app

   ```bash
    npx expo start
   ```
